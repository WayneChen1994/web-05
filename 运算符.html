运算符 算术运算符：+ - * / % 单目运算：n++ --> n = n + 1 当运算符写在前面，先加1再运算； 当运算符写在后面，先运算再加1。 n-- --> n = n - 1 当运算符写在前面，先减1再运算； 当运算符写在后面，先运算再减1。 当不同类型的数据参与运算时，内部首先会进行隐式转换，转换结束之后，再进行计算。 关系运算符： == != > < >= <= === 注意：使用==，内部会进行一个隐式类型转换，转换完毕之后再进行比较。===则不进行类型转换，直接比较。 当number与number类型比的时候，直接比较值的大小； 当String与String类型比的时候，比较ASCII值。 当number与String比的时候，会试图将String转为number再进行比较。 逻辑运算符： && 与运算 || 或运算 ! 非运算 与Python相同，也遵循短路原则 三目运算符： 格式：关系运算表达式?值1:值2; 说明： 值1与值2可以是表达式，可以是常量，可以是变量。 若关系表达式成立，则返回值1，若不成立，则返回值2。 作用： 二选一作用。 赋值运算： += -= *= /= %=
